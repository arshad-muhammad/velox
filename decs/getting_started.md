# Getting Started with Velox

## Installation

1. Clone the repository.
2. Run `python velox.py` to get started.

## Syntax Overview

### Print Statement

```velox
print("Hello, Velox!");
 ```

## Variables

 ```x = 5;
 ```

## Functions

 ```function greet(name) {
    print("Hello, " + name + "!");
}
 ```

## Loops

 ```for (i = 0; i < 10; i = i + 1) {
    print(i);
}```
